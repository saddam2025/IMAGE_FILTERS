# 🖼️ Image Processing Application in C++

A console-based Image Processing Application developed in C++ as a final project for the CS-213 Object Oriented Programming course.

This application allows users to load images, apply multiple filters and effects, then save the edited images using a simple menu-driven interface.

---

# 📌 Features

The application supports the following image filters and operations:

1. Grayscale Filter
2. Black & White Filter
3. Invert Colors
4. Merge Two Images
5. Flip Image
   - Horizontal
   - Vertical
6. Rotate Image
   - 90°
   - 180°
   - 270°
7. Lighten / Darken Image
8. Crop Image
9. Add Frames
   - Simple Frame
   - Fancy Frame
10. Edge Detection
11. Resize Image
12. Blur Filter
13. Load New Image
14. Save Image

---

# ▶️ How to Run

## 1. Clone the Repository

```bash
git clone https://github.com/saddam2025/Repo-OOP.git
```

---

## 2. Open the Project

Open the project using:

- Visual Studio
- VS Code
- Code::Blocks
- Any C++ IDE

---

## 3. Build and Run

Compile and run the program.

Example using g++:

```bash
g++ main.cpp -o app
./app
```

---

# 📋 Program Workflow

1. Enter the image file name.
2. Choose a filter from the menu.
3. Apply edits to the image.
4. Save the edited image.
